# hybrid
Matlab implementation of hybrid classifier.
The "hybrid‚Äù classifier is a linear classifier. It replaces the negative samples by a prior, and then finds a hyperplane which separates the positive samples from this prior.

This repository includes the implementation, with a short demo.
